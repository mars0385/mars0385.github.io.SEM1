Tags in HTML

Unordered lists
The "ul" tag creates a list with no particular order.
Ordered lists
The "ol" tag creates a list with a certain order in mind.
Description lists
The "dl" tag creates a list where you can add a description to the content.
Heading tags
The tags h1-h6 are used to create headings on the html page.
Section tag
The section tag allows you to section off your html page into different parts for organization.
